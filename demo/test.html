rModal - a responsive modal window for desktop or mobile written in javascript

rModal is a responsive modal window for desktop or mobile. It can display any type of content; text, video, images, iframes, and its content can be supplied with inline markup or as JSON data by an AJAX request.

rModal does not require any framework to work and is compatible with all major modern browsers from Internet Explorer 10 to the last version of Chrome.
